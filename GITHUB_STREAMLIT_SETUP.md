# GitHub and Streamlit Setup Guide for ChurnGuard

## Upload to GitHub Using Browser

1. Create a new GitHub repository named `ChurnGuard`.
2. Choose Public.
3. Do not add a README if you are uploading this package.
4. Click Add file → Upload files.
5. Upload all files and folders from this project.
6. Commit changes.

## Deploy on Streamlit Cloud

1. Go to Streamlit Community Cloud.
2. Click New app.
3. Select repository: `YOUR-USERNAME/ChurnGuard`.
4. Branch: `main`.
5. Main file path: `app.py`.
6. Click Deploy.

## Resume Bullet
